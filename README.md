# LG COE Voting Backend

### 개발 환경

```
- **NodeJS:** `v18.18.1` (LTS)
- **PackageManager:** `npm`

```

## Getting Started

```bash
# Modules install
npm install

# Server start
npm run {key}

"start": "export NODE_ENV=dev && nodemon ./bin/www"
"start-local": "export NODE_ENV=local && pm2 start ./bin/www --name local"
"restart-local": "export NODE_ENV=local && pm2 restart local"

```
